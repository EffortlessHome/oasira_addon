# Oasira Cloud Bridge Add-on

Connect your Home Assistant instance to Oasira Cloud services.

## About

This add-on provides:
- Cloudflare Tunnel connectivity for secure remote access
- Matter Hub integration for smart home device management
- Web dashboard for monitoring and configuration

## Configuration

```yaml
email: your-email@example.com
password: your-password
system_id: your-system-id (optional if you have only one system)
ha_url: http://homeassistant.local:8123
dashboard_port: 8080
```

## Installation

1. Add this repository to your Home Assistant add-on store
2. Install the "Oasira Cloud Bridge" add-on
3. Configure your credentials
4. Start the add-on
